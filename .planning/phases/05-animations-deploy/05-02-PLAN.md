## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-03 | Spoofing | DNS/domain hijacking | transfer | Domain DNS managed by Cloudflare — Cloudflare handles DNSSEC and domain verification. User responsibility to secure Cloudflare account with 2FA. |
| T-05-04 | Tampering | Response header injection | mitigate | public/_headers sets X-Frame-Options: DENY, X-Content-Type-Options: nosniff, Referrer-Policy, Permissions-Policy to prevent clickjacking, MIME sniffing, and information leakage |
| T-05-05 | Information Disclosure | Server version headers | transfer | Cloudflare strips server version headers by default on Pages. No action needed. |
| T-05-06 | Denial of Service | Traffic flooding | transfer | Cloudflare provides DDoS protection on free tier. Static site with no origin server eliminates backend DoS vector. |
