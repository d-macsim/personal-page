**Step 2: Create `src/components/HeroSection.tsx`**

Create a React component that serves as a Motion v12 animated island. The component must:

1. Import `motion` and `useReducedMotion` from `"motion/react"` (NOT from `"framer-motion"`)
2. Call `useReducedMotion()` to get `shouldReduce` boolean
3. Create a helper function `item(delay: number)` that returns Motion props:
   - `initial`: if `shouldReduce` is true, `{ opacity: 1, y: 0 }` (no animation); otherwise `{ opacity: 0, y: 20 }`
   - `animate`: `{ opacity: 1, y: 0 }`
   - `transition`: `{ duration: 0.6, delay, ease: [0.16, 1, 0.3, 1] }`
4. Render a `<section>` with:
   - `aria-label="Hero"`
   - Classes: `relative flex flex-col items-center justify-center min-h-[calc(100dvh-5rem)] text-center px-4`
   - The `min-h-[calc(100dvh-5rem)]` accounts for BaseLayout's `pt-20` (5rem) nav offset (per D-01)
5. Inside the section, render in order:
   a. **Glow element** (not animated by Motion, uses CSS class `glow-pulse`):
      - Outer div: className="absolute inset-0 overflow-hidden pointer-events-none", aria-hidden="true"
      - Inner div: className="glow-pulse absolute top-1/4 left-1/2 -translate-x-1/2 -translate-y-1/2 w-[60%] h-[40%] rounded-full"
      - Inner div style: background radial-gradient(ellipse at center, rgba(99,102,241,0.20) 0%, transparent 70%)

   b. **Name** (per D-05, D-07 delay 0ms):
      - `<motion.h1 {...item(0)} className="font-bold" style={{ fontSize: "var(--font-size-display)", lineHeight: 1.1 }}>Dragos Macsim</motion.h1>`

   c. **Title** (per D-05, D-07 delay 150ms):
      - `<motion.p {...item(0.15)} className="mt-4" style={{ fontSize: "var(--font-size-heading)", color: "var(--color-text-muted)" }}>AI Specialist & Product Builder</motion.p>`

**Step 3: Add glow-pulse CSS keyframes to `src/styles/global.css`**

Add the following AFTER the existing `@layer base { ... }` block (do NOT modify existing content):

```css
/* ---- Glow Pulse Animation (Phase 2 Hero) ---- */
@keyframes glowPulse {
  0%, 100% { opacity: 0.12; }
  50% { opacity: 0.22; }
}

.glow-pulse {
  animation: glowPulse 4s ease-in-out infinite;
}

@media (prefers-reduced-motion: reduce) {
  .glow-pulse {
    animation: none;
    opacity: 0.16;
  }
}
```
