| T-02-06 | Information Disclosure | JSON-LD sameAs URLs | accept | LinkedIn/GitHub profile URLs are intentionally public information |
| T-02-07 | Tampering | astro.config.mjs site property | accept | Static configuration value, no runtime modification possible |
| T-02-08 | Spoofing | OG meta tags | accept | OG crawlers validate against the actual page URL, no security impact for a static portfolio |
