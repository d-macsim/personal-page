### Unsatisfied Requirements Detail

#### EXP-01: Experience timeline with key achievements (PARTIAL)

**Success Criterion:** "An experience timeline lists all three roles with company name, date range, and at least two key achievements each"

- `src/data/cv.ts` defines `achievements: string[]` with 2-3 bullets per role
- `src/components/CVSection.astro` maps roles to `{ body: [] }` — hardcoded empty array
- `src/components/TimelineColumn.astro` declares `body: string[]` in interface but never renders it
- Result: Company, title, and dates display correctly, but achievement bullets are silently dropped

#### CONT-02: Contact reachable from any position (PARTIAL)

**Success Criterion:** "A persistent navigation element gives access to the contact section from any scroll position"

- Nav works correctly on the homepage (all hash links scroll to sections)
- After ClientRouter navigation from `/` to `/now`, persisted nav retains `#about` hrefs
- Click handler intercepts hash links and calls `querySelector('#about')` which returns null on `/now` DOM
- Direct browser load of `/now` works (server renders `/#about` hrefs which bypass handler)
